Software Requirements:
GL, glut, GLU, glfw, GLEW
How to download from GitHub: 
1. click on the included link in the document
2. select each file and download RAW as save them into their respective directories.
Or if you have the zip, uncompress the zip.

How to run:
1. Install the required libraries if you haven't yet.
2. navigate to main
3. run g++ main.cpp -o main -lGL -lglut -lGLU -lglfw -lGLEW
4. follow the previous command with ./main
